2.1 Some Terminology

Inputs x and Outputs y vectors are connected by a matrix W that defines the Weights between i and j nodes that represent elements from each vector respectively, which is called a neuron. A neuron is fired by an Activation Function g(·). To validate the output function y(x,W) values are used Target vectors that contain the expected output, both target and output vectors are dimensionally equal to each other. The Error is computed by E(y,t), representing inaccuracies from the system.

2.2 Testing Machine Learning Algorithms

Learning is based on training to get better-predicted outputs, this means that algorithms need to verify if the output generated matches the testing reliable data provided by the target vectors. The error E can express how trained the machine is.

The target vectors do not handle alone all the possible outputs, once we want the algorithms to generalize examples that were not seen already. To reach this objective, a Test Set is needed. This vector does not change weights or any other parameters of the system.

2.2.1 Overfitting

When an algorithm over-train with the target vectors, it will not be able to generalize due to it capability to learn about inaccuracies and noise included on the data that is given.

To skip this problem, we need a third set o data that is called Validation Set, which will be used to stop the learning process as soon as the model reaches almost a perfect generalization. This process is known as Cross-validation in statistics.
